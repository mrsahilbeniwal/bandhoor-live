# Bandhoor

**Where Celebrations Find Their Perfect Match**

A premium wedding decision platform connecting couples with curated wedding vendors.

## Tech Stack

- **Frontend**: React + Vite
- **Backend**: Node.js + Express (coming soon)
- **Database**: MongoDB

## Quick Start

```bash
cd client
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
